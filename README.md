# Machine Learning 1 - Fundamentals & Ensemble Models

This repository contains two comprehensive project developed for **Machine Learning 1** course (BI-ML1) at **FIT CTU in Prague**.\
This projects cover full ML pipeline:
- Exploratory Analysis
- Data Preprocessing
- Models Training and Selection
- Custom implementation
- Final Evaluation

## 🚀 Project Overview

### [Task 1: Regression & Model Selection](./01/)
A comparative study of regression techniques to predict continuous target.
* **Models:**
  * Decision Tree,
  * kNN (k-Nearest Neighbors),
  * Linear Regression,
  * Ridge Regression.
* **Key Focus:**
  * Hyperparameter tuning
  * Final model validation of unseen evaluation data

### [Task 2: Classification & Ensemble Learning](./02/)
An exploration of classification metrics and the power of ensemble models.
* **Models:**
  * Custom Random Forest implementation (compared to scikit one)
  * Decision Tree
  * kNN (k-Nearest Neighbors)
  * AdaBoost
  * Logistic Regression
* **Key Focus:**
  * Deep understanding of ensemble models (gained by custom implementation)
  * Performance visualization (ROC/AUC, F1-Score, Confusion Matrix, ...)
